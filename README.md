## 🚀 사용 방법

### 1. 로컬 테스트

1. **파일 다운로드**
   - 화면 하단 **"Download files"** 버튼 클릭
   - ZIP 파일 다운로드

2. **압축 해제**
   - ZIP 파일을 원하는 폴더에 압축 해제

3. **브라우저에서 열기**
   - `index.html` 파일을 더블클릭
   - 또는 웹 브라우저로 드래그 앤 드롭

4. **확인사항**
   - ✅ 로고 인트로 애니메이션 (3초)
   - ✅ 모든 섹션 정상 표시
   - ✅ 네비게이션 메뉴 작동
   - ✅ 문의 폼 제출 테스트
   - ✅ 파비콘 표시 (브라우저 탭)
